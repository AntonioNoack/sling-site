slingfeature:attach-features

Full name:

org.apache.sling:slingfeature-maven-plugin:0.2.0-SNAPSHOT:attach-features

Description:

Attach the feature as a project artifact.

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: test.
  • The goal is thread-safe and supports parallel builds.
  • Binds by default to the lifecycle phase: package.

Optional Parameters

Name Type Since Description
feature String - Inlined model. This parameter is evaluated in the DependencyLifecycleParticipant.
featureFile File - The feature file.. This parameter is evaluated in the DependencyLifecycleParticipant.
Default value is: ${basedir}/src/main/osgi/feature.json.
jarStartLevel int - The start level for the attached jar/bundle. This parameter is evaluated in the DependencyLifecycleParticipant.
Default value is: 20.
skipAddFeatureDependencies boolean - If set to true the artifacts from the feature are not as dependencies to the project. This parameter is evaluated in the DependencyLifecycleParticipant.
Default value is: false.
skipAddJarToFeature boolean - If set to true the main jar artifact is not added to the feature. This parameter is evaluated in the DependencyLifecycleParticipant.
Default value is: false.
skipAddJarToTestFeature boolean - If set to true the main jar artifact is not added to the test feature. This parameter is evaluated in the DependencyLifecycleParticipant.
Default value is: false.
skipAddTestFeatureDependencies boolean - If set to true the artifacts from the test feature are not as dependencies to the project. This parameter is evaluated in the DependencyLifecycleParticipant.
Default value is: true.
testFeature String - Inlined test model. This parameter is evaluated in the DependencyLifecycleParticipant.
testFeatureFile File - The test feature file.. This parameter is evaluated in the DependencyLifecycleParticipant.
Default value is: ${basedir}/src/test/osgi/feature.json.
